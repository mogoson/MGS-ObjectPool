# MGS-ObjectPool
- [中文手册](./README_ZH.md)

## Summary
- Unity object pool.

## Environment
- Unity 5.0 or above.
- .Net Framework 3.5 or above.

## Achieve
- Generic object pool.
- Recycling objects that need to be repeatedly created and destroyed.
- Manage gameobject pool.
